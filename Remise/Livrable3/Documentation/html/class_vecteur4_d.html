Detailed Description

template<class T>
class Vecteur4D< T >

Classe generique pour implanter des vecteurs en 4D.

Cette classe generique est definie pour les vecteurs de dimension 4. Les operations necessaires sont definies, soient:

  • les constucteurs
  • operateur d'egalite
