Detailed Description

template<class T = double, const int N = 3>
class Vecteur< T, N >

Definition d'une classe generique pour les vecteurs.

Cette classe generique implante des vecteurs en n dimensions. Le type du vecteur est aussi generique (double, float, int,...). Les fonctions communes definies sont:

  • [] pour la lecture et l'ecriture
  • conversion double/float ou int/float
  • norme du vecteur
  • si le vecteur est nul
  • operateur =
  • operateur ==
  • operateur +=
  • operateur -=
  • normalisation du vecteur
  • operateur *= (multiplication par un scalaire)
  • operateur /= (division par un scalaire)
  • operateur +
  • operateur -
  • operateur * (multiplication par un scalaire)
  • operateur / (division par un scalaire)
  • produit scalaire
  • la reflexion du vecteur u par rapport a l'espace vectoriel orthogonal
  • projection du vecteur u sur l'espace vectoriel orthogonal
  • projection du vecteur u sur une droite
  • operateur << (sortie texte)
  • operateur >> (entree texte)
  • operateur < (sortie binaire)
  • operateur > (entree binaire)
