Detailed Description

Author
Sacha L-Roussel

Classe qui définit l'interface d'édition du programme; Celle-ci permet de créer et d'éditer des cartes de jeux composée de Station Spatiale, de vaisseaux, d'astéroïde, de barrière et de bonus.
