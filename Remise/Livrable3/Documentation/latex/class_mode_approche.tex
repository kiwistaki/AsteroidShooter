\subsection{Detailed Description}
La classe \hyperlink{class_mode_approche}{Mode\-Approche} permet d'implementer le mode d'approche du vaisseau virtuel. 

Cette classe contiendra des méthodes qui le changement d'etats, un vecteur des elements de jeu qui seront des obstacles et l'asteroide cible.
