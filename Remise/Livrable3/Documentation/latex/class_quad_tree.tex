\subsection{Detailed Description}
Cette class implémente la fonctionnalité d'un \hyperlink{class_quad_tree}{Quad\-Tree}. 

C'est un arbre dans lequel chaque noeud contient exactement 4 enfants. On ajoute des noeuds supplémentaires en ajoutant des éléments dans l'arbre\-: lorsque le nombre d'objets se trouvant dans un noeud est supérieure à une constante définie par l'utilisateur, on subdivise le noeud courant en 4 sous-\/noeuds.

Cet arbre permet de traiter les collisions seulement avec les autres objets qui se trouvent dans le même noeud, réduisant dramatiquement le nombre de cas à traiter.

L'insertion utilise des boites englobantes non-\/orientées pour déterminer si un objet est dans un noeud. Cette méthode n'est pas la plus précise, mais avec un \hyperlink{class_quad_tree}{Quad\-Tree} on cherche a donner une approximation rapidement pour éviter de plus gros calculs, donc une boite englobante non-\/orientée suffit largement.
