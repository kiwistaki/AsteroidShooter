Detailed Description

Author
Sacha L-Roussel

Classe définissant la barre d'outils lattérale de la fenêtre d'édition. Elle contient un certain nombre de boutons permettant d'accéder à des fonctions autrement dans la barre de menu de la classe InterfaceEdition.
