\subsection{Detailed Description}
La classe \hyperlink{class_mode_attaque}{Mode\-Attaque} permet d'implementer le mode d'attaque du vaisseau virtuel. 

Cette classe contiendra des méthodes qui le changement d'etats, un vecteur des elements de jeu qui seront des obstacles et l'asteroide cible.
